Invokes the correct function in this grouped function based on the values provided. If no function is found with the appropriate arguments for the input values then an exception is thrown.

Parameters
valuesThe values to pass to the function to be called
return_valueThe return value for this function.
objectThe object this function was invoked on. Set to NULL if no object was provided.
Exceptions
std::logic_errorThrown if no function can be found that can deal with the values provided.
